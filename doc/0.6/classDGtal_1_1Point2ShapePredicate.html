Detailed Description

template<typename TSurface, bool isUpward, bool isClosed>
class DGtal::Point2ShapePredicate< TSurface, isUpward, isClosed >

Aim: Predicate returning 'true' iff a given point is in the 'interior' of a given shape, 'false' otherwise.

Template Parameters
TSurfacea type that is a model of COrientableHypersurface. Must separate the space in two disjoints parts (the 'interior' and the 'exterior') and must be able to return for any given point (of type TSurface::Point) the signed distance to itself (of type TSurface::Distance) by a method called signedDistance()
isUpwarda bool for the orientation ('true' means that the interior is the set of points of positive distance)
isCloseda bool for the closure ('true' means that the surface is included) 
 For instance, in 2d, you can define four shapes from one straight line of equation \form#312:
  • the set \( \{(x,y) | y > ax + b \} \)
  • \( \{(x,y) | y \geq ax + b \} \)
  • \( \{(x,y) | y < ax + b \} \)

  • \( \{(x,y) | y \leq ax + b \} \)
