Detailed Description

Modifier class in a Board2D stream. Useful to choose your own style for a given class. Realizes the concept CDrawableWithBoard2D.

Examples:
io/boards/dgtalBoard2D-3-custom-classes.cpp, io/boards/dgtalBoard2D-3-custom-points.cpp, io/boards/dgtalBoard2D-4-colormaps.cpp, logoDGtal.cpp, topology/ctopo-1.cpp, and topology/ctopo-2.cpp.
